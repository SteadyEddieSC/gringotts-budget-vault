# Gringotts Budget Vault

Private, local-first household budget PWA. v59 is the first modular Cloudflare Pages/GitHub release.

## Deploy

Use Cloudflare Pages, not Workers/Wrangler:

- Project type: Pages / Static
- Repository: `SteadyEddieSC/gringotts-budget-vault`
- Production branch: `main`
- Build command: leave blank
- Output directory: `/`

## Repository boundary

The repository contains app code only. Keep personal exports, backups, screenshots, and spreadsheet files outside this repo.
