# Gringotts Budget Vault v61

## Focus

Rules Engine II.

## Changes

- Added rule aliases for alternate merchant names.
- Added rule priority. Lower numbers run first.
- Added enable and disable controls for rules.
- Added rule hit counts.
- Added conflict review for transactions that match more than one rule.
- Added Apply rules now.
- Updated app shell and service worker cache to v61.

## Use

Open the Rules tab, add a merchant match, optional aliases, category, owner, and priority. Use the conflict panel to tune overlapping rules.
